Установка Python на разных системах

Windows

Для установки Python на системе Windows нужно просто перейти на официальный сайт на страницу загрузок

Скачиваем последнюю версию (Или ту, которая приглянулась)

Запускаем файл и обязательно нажимаем на Add python.exe to PATH

Нажимаем Install now

Ждём установки

После завершения установки закрываем программу инсталляции

Перезагружаем компьютер

Проверяем, скачан ли Python

Нажимаем на клавиатуре сочетание клавиш win+r

Вводим в поле ввода powershell

Нажимаем клавишу Enter

Пишем в поле ввода python --version (команда для проверки версии Python)

Нажимаем Enter

Если у Вас вывелась версия и никаких ошибок нет, то поздравляю! Вы успешно установили Python!

Linux

Ubuntu

Вероятнее всего, Python у Вас уже есть по умолчанию, но если это не так, то Вам нужно проосто открыть терминал ctrl+alt+t

Написать sudo apt update && sudo apt upgrade -y

После чего ещё sudo apt install python3 -y

После установки нужно написать python3 --version (проверка версии питона)
